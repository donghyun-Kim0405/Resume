- 기간 : 2021.01 - 2021.08
- 언어 : Java, C++, JavaScript
- 관련 기술
  - Android, Retrofit2, OpenCV, Coroutine

- 역할 (네이티브에서의 전반적인 기능을 대부분 담당하였습니다.)
  - 영상 녹화기능 구현

  - Retrofit을 이용한 서버 통신 기능 구현(영상전송, 충돌 정보 전송)
  - 서버단 영상 추출기능 구현(끼어들기 차량 인식 및 영상추출, 충돌값 기반 영상 추출)
